Introduccion

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web.
Se compone de una serie de etiquetas que definen la estructura y el contenido de una página web.

Estructura Básica

Una página HTML básica consta de las siguientes partes:

  • La etiqueta ``, que engloba todo el contenido de la página.

  • La etiqueta ``, que contiene metadatos como el título de la página y enlaces a archivos CSS y JavaScript.

  • La etiqueta ``, que contiene el contenido visible de la página, como texto, imágenes y otros elementos.
